🎨 Protótipo UI/UX - Portal de Notícias Academia Venum
Este repositório contém a documentação visual e o planejamento de interface para o Portal Academia Venum, desenvolvido no Figma. O projeto visa integrar uma interface moderna e intuitiva a um sistema robusto de entrega de conteúdo para entusiastas de fitness e saúde.

📌 Sobre o Projeto
O protótipo foca na organização estrutural de um portal dinâmico, priorizando a hierarquia visual e a escaneabilidade. Foi projetado para ser totalmente responsivo, garantindo uma experiência fluida em desktops, notebooks e dispositivos móveis.

Destaques da Prototipagem:
Hierarquia de Conteúdo: Uso de tipografia e pesos visuais para destacar eventos principais, como o "Crossfit Revolucionário".

Layout de Coluna Dupla: Área principal dedicada à leitura detalhada e barra lateral "Veja Também" para aumentar a retenção do usuário.

Navegação Estruturada: Menu superior com seções institucionais (Sobre Nós, Fale Conosco) e integração com redes sociais no rodapé.

Foco em UX: Inclusão de elementos de usabilidade, como links de retorno ("Voltar") e botões de ação (CTA) padronizados.

🛠️ Tecnologias e Ferramentas
Design/UI: Figma (Prototipagem de alta fidelidade).

Implementação (Stack Planejada): PHP, HTML5, CSS3, JavaScript e MySQL.

Controle de Versão: Git e GitHub.
